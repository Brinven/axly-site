# Lessons — Axly.com Redesign

## Assets & Graphics
- **No emoji as design elements.** Replace all emoji (🎨, ⛏, etc.) with custom graphics or SVG icons. Flag any spot where a custom image would elevate the design — Mike has extensive local image gen capability and can produce assets quickly.
- When in doubt, request a custom graphic rather than reaching for a generic icon library. Showcasing the image gen work IS the brand.
